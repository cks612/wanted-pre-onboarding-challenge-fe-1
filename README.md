## 아쉬운점

1. 시간문제

- 급하게 과제를 하다보니 시간에 쫓겨 스타일과 기능구현을 제대로 못 한게 아쉽습니다.

2. useMutation

- useMutation에는 파라미터로 하나이상을 보내고 싶으면 객체로 만들어서 보낼 수 있다는 것을 이 과제를 하면서 알 수 있었습니다. 완벽하게 이해하지 못해서 아쉽지만 이번 챌린지를 통해 더욱 더 성장하고 싶습니다.
